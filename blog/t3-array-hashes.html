Arrays and Hashes

02/03/2015

Arrays

An array is a list. It allows you to use a method on many things at a time without listing them all out individually. Each piece of an array has a number associated with that order but it starts with 0 rather than 1.

Hashes

Similar to an array, a hash is basically a list. However, each piece of the hash has two parts: a key and a value. These keys and values are partnered together.
